$definedIn=pthread.h
$funcName=pthread_create
$explanation=Create a new thread with specified attributes and start routine.

tid|pthread_t *restrict|thread ID pointer to store thread ID
tattr|const pthread_attr_t *restrict|pointer to thread attribute structure
start_routine|void *(*)(void *)|function pointer to the code the thread will run
arg|void *restrict|argument to be passed to the start routine

0->thread created successfully
EAGAIN->system limit on threads was reached
EINVAL->invalid attribute settings in tattr
EPERM->insufficient permission to set scheduling attributes
